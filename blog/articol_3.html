Primii pași practici: instalarea VS Code, extensii și GitHub

În primele articole am vorbit despre motivație și set-up. Acum intrăm în practică: instalăm uneltele și le configurăm corect ca să putem lucra rapid.

Instalarea Visual Studio Code

Mergi pe site-ul oficial și descarcă varianta potrivită pentru sistemul tău (Windows / macOS / Linux).

Windows (recomandat „User Installer x64”)

  1. Rulează fișierul .exe.
  2. Acceptă termenii și condițiile.
  3. Lasă locația implicită pe C: (recomandat pentru update-uri automate și pentru a evita probleme de permisiuni).
  4. La „Additional tasks” bifează:
    • Add “Open with Code” în meniul contextual (fișiere și foldere).
    • Add to PATH (ca să poți rula code din Command Prompt/PowerShell).
  5. Install → Finish (poți lăsa „Launch VS Code” bifat).

macOS

  1. Descarcă fișierul .zip și dezarhivează-l.
  2. Mută Visual Studio Code în folderul Aplicații (Applications).
  3. Deschide aplicația (poate apărea un mesaj de la macOS, apasă OK).
  4. Ca să poți rula code din Terminal: Command+Shift+P → „Shell Command: Install 'code' command in PATH” → Enter.

Linux (Ubuntu/Debian)

  1. Descarcă pachetul .deb.
  2. Instalează-l cu dublu-click sau din terminal cu: sudo dpkg -i nume_pachet.deb
  3. Dacă apar erori de dependențe, rulează: sudo apt --fix-broken install

Setările de bază (ușoare și utile)

Deschide Settings:

  • Windows/Linux: Ctrl + ,
  • macOS: Command + ,

Recomandări:

  • Theme: „Dark+ (default)”.
  • Editor > Font Size: 14–16 (confort vizual).
  • Editor > Word Wrap: on (să nu iasă liniile în afara ecranului).
  • Editor > Tab Size: 2 și Insert Spaces: on.
  • Files > Auto Save: onFocusChange (salvează când schimbi fereastra).
  • Language: recomand engleza (găsești mai ușor soluții pe internet și te obișnuiești cu termenii).

Extensii esențiale

Deschide Extensions (Ctrl/Cmd + Shift + X) și caută/instalează:

  1. Live Server (Ritwick Dey)

    Click dreapta pe index.html → Open with Live Server (se deschide în browser și se reîncarcă la salvare).

  2. Prettier – Code formatter (Prettier)

    După instalare, setează-l ca Default Formatter:

    • Command Palette (Ctrl/Cmd + Shift + P) → „Format Document With…” → „Configure Default Formatter…” → Prettier.
    • Asigură-te că „Format On Save” este activ.
  3. Auto Rename Tag

    Când modifici un tag HTML, se modifică și tag-ul de închidere.

  4. Color Highlight

    Afișează culoarea în cod când folosești coduri hex, rgb, hsl etc.

  5. GitLens

    Îmbunătățește suportul Git în VS Code (vizualizare istoric, blame etc.).

  6. ESLint

    Verifică și aplică regulile de stil pentru JavaScript (necesită un fișier .eslintrc în proiect).

Git și GitHub (cont + conectare)

Git este un sistem de control al versiunilor, iar GitHub este o platformă online care găzduiește depozite Git. Împreună, acestea îți permit să gestionezi și să colaborezi la proiecte de cod.

Creează/folosește cont GitHub

  1. Creezi contul aici → Sign up (user, email, parolă).
  2. Confirmă emailul.

Conectează VS Code cu GitHub (pentru sync & repo)

  • În VS Code, jos stânga (iconița cu rotiță) → Turn on Settings Sync → alege „Sign in with GitHub” → autorizează.
  • Bifează ce să sincronizeze (setări, extensii, keybindings).

Verifică Git local

  • Deschide terminalul integrat (Ctrl/Cmd + `) și scrie: git --version
  • Dacă apare versiunea, e instalat. Dacă nu, instalează-l de pe git-scm.com.

Primul repository (rapid)

  1. Creează repo pe GitHub (New → nume, Public/Private).
  2. În VS Code, deschide folderul proiectului tău.
  3. Deschide terminalul integrat (Ctrl/Cmd + `) și inițializează Git: 
    git init
git add .
git commit -m "Prima versiune"
git branch -M main
git remote add origin https://github.com/<username>/<repo>.git
git push -u origin main
    (înlocuiește <username> și <repo> cu datele tale)

Alternativ, din Source Control (icon cu ramificații) → Publish to GitHub (dacă ești logat în VS Code).

Gata atelierul

Acum ai:

  • VS Code instalat și setat.
  • Extensii de bază.
  • Cont GitHub și repo public.
